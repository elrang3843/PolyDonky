# PolyDoc
HWP,HWPX,DOC,DOCX,HTML, HTM, md 파일을 읽어서 IWPF라는 독자 형식으로 저장하는 에디터 앱

* 개발 언어 : C# + WPF(또는 WinUI 3)
* 파일 포맷 관련 파일 : IWPF.md

1. 주요 기능
   1) iwpf, hwp,hwpx,doc,docx,html, htm, md, txt 문서 읽기
   2) iwpf, hwpx, docx, html, htm, md, txt 형식으로 저장 가능
   3) 편리한 문서 편집 기능
   4) 다양한 테마 구성
   5) 다국어 지원(한글, 영어)

2. 메뉴 구성
   1) 파일
      - 새 파일 : 기본 IWPF 모드
      - 불러오기 : 기본 IWPF, md, txt - 다른 형식 저장은 외부 모듈을 실행해서 처리
      - 저장 : 기본 IWPF, md, txt - 다른 형식 저장은 외부 모듈을 실행해서 처리
      - 다른이름으로 저장 : 기본 IWPF, md, txt - 다른 형식 저장은 외부 모듈을 실행해서 처리
      - 미리보기 : 편집용지, 인쇄 색상 설정 포함
      - 인쇄
      - 닫기 : 현재 편집 중인 문서 닫기 - 닫기 전 저장 여부 물어볼 것
      - 종료
   2) 편집
      - 복사
      - 잘라내기
      - 지우기
      - 붙여넣기
      - 가져오기 : 외부의 문서나 개체를 현재 편집 위치에 삽입
      - 내보내기 : 편집창의 선택된 영역을 외부 문서나 개체로 저장하기
      - 문서정보 : 문서 정보 보기 및 작성/편집 정보 편집, 암호 걸기, 워터마크
      - 찾기
      - 바꾸기
   3) 입력
      - 글상자 : 사각형, 말풍선, 구름풍선(머릿속 생각 표현), 가시풍선(번뜩이는 아이디어 표현), 번개상자(팍 와닿는 표현 전달)
      - 표(시트) : 엑셀 같은 표 시트
      - 그래프 : 꺽은선 그래프, 파이 그래프, 막대 그래프, 분포 그래프 등의 그래프
      - 특수문자 : 문자표(특수문자 우선 표시)
      - 수식 : 특수 문자 중 수학, 과학에 많이 쓰이는 문자
      - 이모지 : 이모지 또는 이모티콘 이미지
      - 도형 : 직선, 폴리곤선, 스프라인 선, 정사각형, 직사각형, 사다리꼴 평행사변형, 정삼각형, 이등변 삼각형, 직각 삼각형, 삼각형, 원, 타원, 호, 폴리곤면, 스프라인 면, 화살표
      - 그림 : PNG, BMP, JPEG, TIFF 지원
      - 사인(불러오기)
   4) 서식
      - 글자 서식 : 폰트, 크기, 글자폭(% 단위), 글자 간격(픽셀 단위), 두껍게, 이탤릭, 위첨자, 아래첨자, 밑줄, 중간줄, 윗줄, 테두리, 글자색, 배경색, 줄 색(테두리색)
      - 문단 서식 : 줄 간격, 문단 간격, 들여쓰기, 내어쓰기, 자동번호(Mark down) 
      - 페이지 서식 : 편집 용지, 색상(단색, 16색, 256색, Full칼라), 여백, 머릿글, 꼬릿글, 다단
   5) 도구
      - 설정 : 사용자, 룰러/눈금 보기, 편집용지 보기, 읽고 쓸 형식 선택/해제(HWP, HWPX, DOC, DOCX, HTM, HTML, md, txt), 사용 언어 선택, 테마 선택
      - 사전 : 외부 앱또는 모듈 연결(우선 순위 낮음)
      - 맞춤법 검사 : 외부 모듈 연결(우선 순위 낮음)
      - 사인 만들기 : 외부에 별개의 독립 앱으로 만들어서 실행 - 입력의 사인(불러오기)와 연계됨, 다른 문서 양식에서 도형이나 그림으로 인식될 수 있는 형식으로.
   6) 도움말
      - 사용 방법(메뉴얼) : 언어별로 별도 파일
      - 라이센스 : 써드파티 포함, 언어별로 별도 파일
      - About : 이 앱에 대한 소개(저작권자 정보 포함, 회사 로고 넣을 것)

4. 개발원칙
   1) 내부 저장은 기본적으로 IWPF, md, txt 형식으로만 저장
   2) hwp, hwpx, doc, docx, html, htm 형식 저장 및 읽기 기능은 외부에 Command mode로 수행되는 별도의 컨버터 앱 모듈을 만들거나 채용하여 변환하는 형태로 진행.
   3) 다른 형식으로 저장시 사용자에게 항상 한번 더 확인할 것
   4) 다른 형식(hwp, hwpx, doc, docx, htm, html, md 등)을 불러오면 최대한 원본 문서에 가깝게(99% 목표) 변환 및 보여줄 것.
   5) 사용 연령대를 젊은 층(학생, 청소년, 청년, 장년 까지)으로 보고 색상 테마 구현(테마 선택 가능하게)
   6) 다국어 지원(한글 기본 선택, 우선은 영어까지만)
   7) 지원 OS : 윈도우 10 이상
