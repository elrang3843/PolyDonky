4. 일반 단락

[짧은 단락] 이것은 짧은 단락입니다. 문서의 도입부나 간단한 설명을 제공할 때 주로 사용됩니다.

[긴 단락] 이것은 긴 단락입니다. 상세한 내용을 설명하거나 배경 지식을 제공할 때 사용됩니다. 한국어 웹 타이포그래피에서는 적절한 행간(line-height)과 자간(letter-spacing)이 가독성에 큰 영향을 미칩니다. 일반적으로 본문 텍스트는 16px 내외의 크기를 가지며, 줄 간격은 1.6 정도가 적당합니다. 양쪽 정렬을 적용할 경우 단어 사이의 간격이 불규칙해질 수 있으므로 주의가 필요합니다. 단락과 단락 사이에는 적절한 여백을 두어 문맥의 분리를 명확히 해야 합니다. 이러한 설정들은 독자가 문서를 읽는 동안 피로감을 느끼지 않고 내용에 집중할 수 있도록 돕습니다.

5. 텍스트 서식

HTML은 텍스트의 의미와 형태를 표현하기 위한 다양한 태그를 제공합니다.

  • 굵게 (Strong/B): 이 텍스트는 굵게 표시됩니다.
  • 기울임 (Em/I): 이 텍스트는 기울임꼴로 표시됩니다.
  • 밑줄 (U): 이 텍스트에는 밑줄이 그어져 있습니다.
  • 취소선 (S/Strike): 이 텍스트에는 취소선이 그어져 있습니다.
  • 위첨자 (Sup): E = mc2
  • 아래첨자 (Sub): H2O
  • 형광펜 강조 (Mark): 문서 내의 핵심 키워드를 강조합니다.
  • 코드 인라인 (Code): console.log('Hello World'); 함수를 실행하세요.
  • 키보드 입력 (Kbd): 저장하려면 Ctrl + S를 누르세요.
  • 변수 (Var): 수학 공식에서 x와 y는 변수입니다.
  • 약어 (Abbr): HTML 문서를 작성합니다.
  • 작은 글씨 (Small): 저작권 정보나 부가적인 작은 설명에 사용됩니다.

6. 정렬

[왼쪽 정렬] 기본 정렬 방식입니다. 텍스트가 왼쪽 가장자리에 맞춰 정렬되며 오른쪽 가장자리는 불규칙하게 남습니다.

[가운데 정렬] 텍스트가 중앙에 맞춰 정렬됩니다. 주로 제목이나 중요한 짧은 문구에 사용됩니다.

[오른쪽 정렬] 텍스트가 오른쪽 가장자리에 맞춰 정렬됩니다. 날짜나 서명 등에 주로 사용됩니다.

[양쪽 정렬] 텍스트가 양쪽 가장자리에 모두 맞춰 정렬됩니다. 단어 사이의 간격이 자동으로 조절되어 양쪽 여백이 반듯하게 떨어집니다. 신문이나 출판물에서 자주 볼 수 있는 정렬 방식입니다. 문장이 길어질 경우 가독성을 높이는 데 도움이 됩니다.

13. 코드 블록

Python 예제 (줄 번호 시뮬레이션 적용):

def calculate_fibonacci(n):
    """피보나치 수열을 계산하는 함수"""
    if n <= 0:
        return []
    elif n == 1:
        return [0]
    
    sequence = [0, 1]
    while len(sequence) < n:
        next_val = sequence[-1] + sequence[-2]
        sequence.append(next_val)
        
    return sequence
